Quetion File of KeT-LMS     2023-07-08

  1. 1. Name　question(Id-No).txt
    2.
    • Id : Date, for example
    • No : Number of the 1st big question
    • You can add multiple big questions
        Put a blank line between big questions
        Question number will be increased automatically.
    • [ex] question0615-1.txt, question0615-3.txt
      　  (question0615-1 includes 2 big questions, 01 and 02)
  2. 2. Formats
    3.
    • Q  Don't add the number.
    • (Question Statement)  Add // for multiple lines
    • [1] (Small Question)  Put [] for only one small question.
    • [2]   Next small question,...
    • Sheet   Answer Column
    • [1]   :: Allocation. Add ::-1 if you don't use Maxima for the scoring.
    • [2]
    • Ans   Correct answer
    • [1]
    • [2]
    • Put a blank line for the next question.
  3. 3. An example
    4.
    • Q
    • Answer the following questions.
    • [1] Write the definition of the derivative of sin(2x) and find it
    • [2] Integrate sin(2x) from 0 to fr(pi,2).
    • Sheet
    • [1] =  ::5::-1
    • [2] int(sin(2x),x,0,fr(pi,2))=  ::5
    • Ans
    • [1] lim(z,x,fr(sin(z)-sin(x),z-x))=2cos(2x)
    • [2] 1
  4. 4. Confirmation of correct answers by Maxima
    5.
  5.  (I) Add the following lines.
    • Mxcalc
    • [1] sin(2x) Substitute the string in Maxima converted from KeTMath for i1
    • [2] fr(pi,2) Substitute the string in Maxima converted from KeTMath for i2.
    • f(x):i1   Function definition, assume and declare are available.
    • o1:diff(f(x),x)  Substitue the result for o1.
    • o2:integrate(f(x),x,0,i2)
    • return o1::o2  retirm o1 and o2
  6.  (II) Launch 'toolketmath.cdy', Choose 'Call Maxima for answers' and run.
    • You can obtain the result.
    • The result is a list of [2*cos(2*x),1]
